# ⚙️ Instalação / Install
* Cada pasta / For each folder ( cd web -> npm i ) 
```bash
# Clone 
$ git clone https://github.com/Gabriel-Aparecido03/chat.git
# entre na pasta / enter
$ cd chat
# select the folder
$ cd web
$ cd server
# Instale as dependências / Install dependences
$ npm i install
# Execute a aplicação
$ yarn dev
```
